# Paper Directory

## Files

| File | Purpose |
|------|---------|
| `draft.md` | Working prose draft (primary authoring document) |
| `main.tex` | LaTeX manuscript skeleton (for submission) |
| `definitions.md` | Formal mathematical definitions |
| `proofs.md` | Proven propositions with complete proofs |
| `theorem_sketches.md` | Conjectures and research directions (status-labeled) |
| `claim_ledger.md` | Controls allowed claim strength (proven / synthetic / conjectural / not claimed) |
| `literature.md` | Citation scaffold — entries marked VERIFY must be checked before use |
| `outline.md` | Paper section outline |
| `notes.md` | Research notes and open questions |

## Workflow

1. Write and iterate in `draft.md`
2. Check claims against `claim_ledger.md` before adding to LaTeX
3. Verify citations in `literature.md` before including in `main.tex`
4. Proven results go in `proofs.md`; conjectures stay in `theorem_sketches.md`
5. Compile LaTeX: `pdflatex main.tex` (requires figures in `../figures/`)

## Figures

Generated by `experiments/generate_figures.py`. Output in `figures/` (project root).

## Rules

- No claim in `main.tex` may exceed its status in `claim_ledger.md`
- All VERIFY citations must be resolved before submission
- Limitations section must be complete and honest
